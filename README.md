# 📌 Contexte et Problématique

La gestion des finances personnelles constitue un enjeu majeur dans la stabilité économique individuelle.
Cependant, de nombreuses personnes ne disposent pas d’outils adaptés permettant :

- Le suivi structuré des revenus
- La catégorisation des dépenses
- L’analyse statistique des comportements financiers
- L’obtention de recommandations personnalisées

Dans ce contexte, nous avons développé une application interactive permettant :
- La centralisation des données financières
- Leur visualisation dynamique
- Une analyse intelligente assistée par Intelligence Artificielle
